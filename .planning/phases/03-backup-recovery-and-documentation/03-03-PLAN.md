**Method A (DO Console):** Navigate to the Space settings, find Lifecycle Rules, add a rule that expires objects after 30 days.

**Method B (awscli):** Create a `lifecycle.json` file with a 30-day Expiration rule and apply it using:
```bash
aws s3api put-bucket-lifecycle-configuration \
  --endpoint-url https://nyc3.digitaloceanspaces.com \
  --bucket odoo-prod-backups \
  --lifecycle-configuration file://lifecycle.json
```

Include the lifecycle.json content inline:
```json
{
  "Rules": [
    {
      "ID": "expire-backups-30d",
      "Status": "Enabled",
      "Filter": { "Prefix": "" },
      "Expiration": { "Days": 30 }
    }
  ]
}
```

Also add a verification command to confirm the rule is active:
```bash
aws s3api get-bucket-lifecycle-configuration \
  --endpoint-url https://nyc3.digitaloceanspaces.com \
  --bucket odoo-prod-backups
```
