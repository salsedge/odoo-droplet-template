**run-production.sh** — Production verification orchestration:

```
#!/usr/bin/env bash
set -euo pipefail
```

- Resolve SCRIPT_DIR and ODOOKIT_DIR from script location
- Source .env from ODOOKIT_DIR (if exists)
- Pre-flight checks:
  - team-members.json exists (fail with helpful message if not)
  - INFRA_SSH_HOST is set (fail with message)
  - ADMIN_LOGIN and ADMIN_PASSWORD are set (fail with message)
  - SSH connectivity test: `ssh -o ConnectTimeout=5 -o BatchMode=yes -p PORT USER@HOST "echo ok"` (fail with SSH troubleshooting hint)
- Trap EXIT to always close SSH tunnel: `trap 'bash "${SCRIPT_DIR}/ssh-tunnel.sh" stop 2>/dev/null' EXIT`
- Set `export PROD_ODOO_URL="https://localhost:${TUNNEL_LOCAL_PORT:-8443}"`
- Set `export REPORT=1` to generate HTML report

**Execution stages (strictly ordered per CONTEXT.md):**

Stage 1 — Smoke Tests (BLOCKING):
```
echo "=== Stage 1/5: Smoke Tests ==="
cd "$ODOOKIT_DIR"
bash "${SCRIPT_DIR}/ssh-tunnel.sh" start
npx playwright test --project=production tests/smoke/ || { echo "SMOKE TESTS FAILED — aborting production verification"; exit 1; }
```

Stage 2 — Infrastructure Audit (non-blocking):
```
echo "=== Stage 2/5: Infrastructure Audit ==="
bash "${SCRIPT_DIR}/infra-audit.sh" --host "${INFRA_SSH_HOST}" --port "${INFRA_SSH_PORT:-9292}" --user "${INFRA_SSH_USER:-deploy}" || echo "WARNING: Some infrastructure checks failed (non-blocking)"
```
Note: infra-audit.sh runs via separate SSH session, not the tunnel.

Stage 3 — Odoo Audit (non-blocking):
```
echo "=== Stage 3/5: Odoo Audit ==="
npx playwright test --project=production tests/audit/ || echo "WARNING: Some audit checks failed (non-blocking)"
```

Stage 4 — User Creation (non-blocking per CONTEXT.md: partial failure keeps successes):
```
echo "=== Stage 4/5: User Creation ==="
npx playwright test --project=production tests/production/ || echo "WARNING: Some user creation tasks failed — check report for details"
```

Stage 5 — Backup Verification (non-blocking):
```
echo "=== Stage 5/5: Backup Verification ==="
bash "${SCRIPT_DIR}/verify-backup.sh" || echo "WARNING: Backup verification issues detected (non-blocking)"
```

Final summary:
```
echo ""
echo "=========================================="
echo "  Production Verification Complete"
echo "=========================================="
echo "Report: odookit/reports/index.html"
echo ""
```

Make executable (chmod +x).

**package.json** — Add npm scripts:
- `"verify:prod": "bash scripts/run-production.sh"` — Run full production verification
- `"verify:backup": "bash scripts/verify-backup.sh"` — Run backup verification only

Keep all existing scripts unchanged.
